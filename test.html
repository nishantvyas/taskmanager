Launch Extension - Project Completion Test

Test: Daily Progress Mapping Stops on Project Completion

This test verifies that when a project is marked as complete, the daily progress mapping stops at the completion date.

Test Steps:

  1. Open the Launch Extension below
  2. Create a new project with a future target date
  3. Add some tasks and complete a few to see activity in the matrix
  4. Mark the project as complete using the "Mark Complete" button in settings
  5. Observe that the activity matrix stops showing days beyond the completion date
  6. Hover over the completion date to see the special completion tooltip
  7. Check that the activity stats show the completion date
